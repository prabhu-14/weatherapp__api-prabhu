# 🌦️ Weather App (API Integration)

A simple and beginner-friendly Python project that fetches **real-time weather information** using a weather API. This app allows users to enter a city name and view the current **temperature** and **humidity** instantly.


## 📌 Objective

The goal of this project is to learn how to:

* Work with APIs in Python
* Send HTTP requests using the `requests` library
* Handle JSON data
* Build a basic real-world application

## ✨ Features

🌍 Enter any city name
🌡️ Check current temperature
💧 View humidity percentage
⚡ Receive instant weather updates
☁️ Fetch live weather data from API
📍 Search weather by location
🖥️ Easy-to-use interface

## 🛠️ Technologies Used

* 🐍 Python
* 📡 Requests Library
* ☁️ Weather API

## 📚 Learning Outcomes

By completing this project, you will understand:

* API integration in Python
* Making GET requests
* Parsing JSON responses
* Error handling basics
